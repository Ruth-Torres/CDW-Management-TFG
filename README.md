# CDW-Management-TFG
Efficient Construction and Demolition Waste (CDW) management is a global challenge due to its volume and heterogeneity. Traditional methods, based on manual inspections and physical weighing, are slow and error-prone. This project proposes an automated system to identify and classify waste materials.
